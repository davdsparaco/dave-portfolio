pimp my ikea lampan

lighted coffee table

2013

object developed during a 2 days workshop. Taken an exhisting object (in this case the IKEA Lampan) the goal was to break it down to create an object with the same function (or with an added function) but with a different look. The result is a lighted coffe table made out of plexiglas that thanks to their purposely matt surface, gives the room a relaxing atmosphere.
