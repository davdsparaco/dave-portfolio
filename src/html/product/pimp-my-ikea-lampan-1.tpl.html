pimp my ikea lampan

abat-joure

2012

object developed during a 2 days workshop. Taken an exhisting object (in this case the IKEA Lampan) the goal was to break it down to create an object with the same function (or with an added function) but with a different look. The result is a bed side lamp with a movable diffuser composed by 3 bands, able to give the light beams the desired direction.
